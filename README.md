# 🎹 Heartopia Maestro (Auto Piano Player)

โปรแกรมช่วยเล่นเปียโนอัตโนมัติสำหรับเกม **Heartopia** โดยใช้ไฟล์ MIDI ในการจำลองการกดคีย์บอร์ดด้วยความแม่นยำสูง

## ✨ ฟีเจอร์หลัก
- **GUI ใช้งานง่าย**: ไม่ต้องแก้ไขโค้ดเพื่อเปลี่ยนเพลง
- **ระบบ Transpose**: ปรับคีย์เพลงขึ้น-ลงได้ตามต้องการ
- **Start Delay**: ตั้งเวลารอก่อนเริ่มเล่นเพื่อให้สลับหน้าจอได้ทัน
- **Emergency Stop**: กดปุ่ม `ESC` เพื่อหยุดการทำงานทันที
